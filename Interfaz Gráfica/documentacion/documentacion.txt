<P ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%"><FONT SIZE=3><SPAN LANG="en-US"><U>How
to create a Diagram from a file:</U></SPAN></FONT></P>
<P LANG="en-US" ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%">
<BR>
</P>
<P ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%"><SPAN LANG="en-US">There
are multiple ways to create an image: </SPAN>
</P>
<P LANG="en-US" ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%">
<BR>
</P>
<UL>
	<LI><P ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%"><SPAN LANG="en-US">From
	an already created xml file:  Go to File&gt;Open XML. And select
	your archive. </SPAN>
	</P>
</UL>
<P LANG="en-US" ALIGN=JUSTIFY STYLE="margin-left: 0.5in; margin-bottom: 0in; line-height: 100%">
<BR>
</P>
<UL>
	<LI><P ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%"><SPAN LANG="en-US">Creating
	a new XML file. Go to File&gt;Create XML.  Then you can use the
	editor to create it. </SPAN>
	</P>
</UL>
<P LANG="en-US" ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%">
<BR>
</P>
<UL>
	<LI><P ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%"><SPAN LANG="en-US">From
	a template. Go to Template and select one of the examples XML</SPAN></P>
</UL>
<P LANG="en-US" ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%">
<BR>
</P>
<P ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%"><SPAN LANG="en-US">After
that, click the play button on the top-right corner to generate the
image.</SPAN></P>
<P LANG="en-US" ALIGN=JUSTIFY STYLE="margin-bottom: 0in; line-height: 100%">
<BR>
</P>
